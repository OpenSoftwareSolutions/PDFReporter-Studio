Report State View

The Report State view moves to the front when a report is executed. This view shows:

  • The console which displays the report execution process in real-time.

  • A list of errors that occurs during report execution. Double clicking an error will open error dialog with a detailed stacktrace.

  • Some statistics of report execution like fill size, record count, page count, compilation time, filling time, export time.
