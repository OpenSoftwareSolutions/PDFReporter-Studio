Outline View

In the Outline view we can see all JasperReports elements in a form of a tree. These elements can be managed using context menu where we can find the following operations:

  • Creation of new objects, like Group, Dataset, Field, Parameter, Variable, etc.

  • Removal of objects

  • Standard editor operations like undo, redo, copy, cut, paste

  • User interface actions that help to arrange elements, resize container, order elements, etc.

  • We always have "Show Properties" action that will show properties page for the selected element

  • Actions that help us set properties like PDF508 Tags

Users can move elements from one container node to another, or reorder elements by dragging and dropping within the Outline view. Double clicking an element opens an expression editor, or an editor for objects like Crosstab, Table, and List.
