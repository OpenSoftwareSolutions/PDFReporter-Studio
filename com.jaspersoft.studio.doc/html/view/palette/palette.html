Palette View

This view contains every graphic element we can drag from Palette to the graphics editor. There are two modes for this view, one inside the report designer, and another as a separate view. To open the Palette in a separate view, select Windows → Show View → General → Palette . If you close this view the palette will be available inside the graphic editor.

You can configure the Palette view by right-clicking inside it and opening the context menu, which includes options for big or small icons, the way elements are arranged (column, list, icons only, details), and where the Palette is docked.

There are three ways to create elements from the Palette:

  • Drag and drop element over a report container like a band, when object is dropped this will be the position for the element.

  • Click on an element inside the palette, go to a report container like a band click in the place where you want the top left corner will be, and drag in right-down direction.

  • Drag from Palette to Outline view.
