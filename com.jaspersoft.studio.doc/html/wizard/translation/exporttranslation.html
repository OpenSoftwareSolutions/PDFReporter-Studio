How to publish a Jaspersoft Studio translation

In the tutorial How to translate Jaspersoft Studio we have seen how we can create a new translation, in this tutorial will be explained how that translation can be published on the community site to be visible by the Jaspersoft Studio team and the other users.

To publish a translation project right click on the project and select the option Send Translation

Figure 1. Send Translation action


Since you translation will be posted in the community site tracker you must provide a title and a description for the open post, and then hit Next

Figure 2. Provide a title and a description


At this point you need only to provide your login information and hit Finish, all your translation files will be packed inside a zip file and attached to the open post.

Figure 3. Provide the login data


If you are not registered on the community site you can create an account following this link.
